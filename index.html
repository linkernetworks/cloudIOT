Cloud IoT

A IoT platform as a service product in iotfactory, that took container solution advantages in the both cloud and the end device.

Cloud IoT integrate excellent cloud technologies to provide our user an easy to use platform and with pay by use advantage. Our platform want to make Developer and Maker to focus their innovation than handle the infrastructure hardly.

Remote management support

This is ready in Cloud IoT platform for control and management user’s devices. Linker implement an auto discovery flow for devices to have linker’s IoT agent and quickly connect to platform. We also provide a powerful dashboard for user to manage their device status and event deploy or upgrade services.

Gateway input protocols

Gateway input protocols also include in Linker IoT platform. In the gateway that can run Linker service, we support many protocols like TCP/UDP, HTTP/HTTPS, SNMP, syslog, MQTT… User can design their device to input data to Linker IoT ready Gateway and we will collect the data in the cloud for you to quickly use and analysis.

IoT CICD

IoT CICD is new concept for Continue Integrating the Continue Deploying new version application in an auto and easy way. IoT CICD work with Cloud IoT to ready the DevOps flow for quickly build and test your services in any devices you own. With the Cloud IoT and IoT CICD managed deploy, all the innovation can be more easily and quickly to face to the real world and bring you more revenue.
